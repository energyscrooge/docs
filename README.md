# 에너지해커톤2017

- 2017.04.07~08, 1박2일간 서울대학교에서 진행된 에너지를 주제로 한 해커톤 행사 입니다
- ‘에너지와 라이프스타일 융합’이라는 주제로 에너지 데이터를 활용해 실생활에 필요한 새로운 서비스나 비즈니스 모델을 개발이 목표 입니다
- 서울대 전력연구소와 인코어드 주최, LS산전이 주관, 산업통상자원부, 한국에너지기술평가원, 네이버, 아마존 AWS의 후원으로 진행 되었습니다


# 에너지스크루지

- 에너지스크루지팀은 해커톤 상위 4팀에 선정되어 수상하였습니다
- 발표자료와 제안서를 포함하여 자료를 공유합니다
